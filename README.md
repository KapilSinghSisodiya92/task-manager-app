# Full-Stack Task Management Application

A full-stack task management application featuring a Go REST API backend and a Next.js frontend, backed by a PostgreSQL database.

## Prerequisites
- Docker & Docker Compose
- Go (version 1.20+)
- Node.js (version 18+)

## Getting Started
1. Clone the repository.
2. Copy `.env.example` to `.env` and adjust variables if needed.
3. Start the local database:
   ```bash
   docker compose up -d
